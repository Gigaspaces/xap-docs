Supported Platforms

Requirements

GigaSpaces and are implemented using Java, and require the following:

  • Java version 8, 11 or 17
  • Any operating system that is supported by Java (for example Linux/Unix, Microsoft Windows, and Apple Mac OS/X)

requires Microsoft .NET Framework 4.0 or later. Additional information is provided in the Installation section of the Guide.

For information on VMWare support, refer to the VMWare guidelines.

Limitations

The following issue has been encountered in a specific use case:

  • SUSE-10 Linux Enterprise Service Pack 3 has bugs that make the OS network layer unreliable. Do not use this operating system with GigaSpaces products.

Java 9 and later is not currently supported with this version of and due to changes in the Java platform. It is planned to be supported in a future release.
