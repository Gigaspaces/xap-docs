Smart DIH (Kubernetes) - Quick Start Guide

This is a checklist to aid the process of installing (via Kubernetes) and operating Smart DIH.

Prerequisites - Generic

  • Ensure you have a SoR (e.g. Database) up and running. For example, Oracle or MSSQL.

  • Make sure the SoR has tables that contain some data.

Installations

  1. Install Smart DIH using Kubernetes orchestration. FYI: Read our overview for deployment of Smart DIH Orchestration.

  2. Install the Data Integration Agent relevant for the SoR that is being used.
