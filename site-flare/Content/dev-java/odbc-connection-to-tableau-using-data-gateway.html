ODBC Connection to Tableau using Data Gateway

Instructions:

Install ODBC Driver for PostgreSQL (tested on v10.03) to be used by Tableau.

Open Tableau (tested on v2020.4)

In the left panel where the title is "To a Server", choose "More..." and select "Other Databases (ODBC)"

In the dialog, choose "Driver" and then choose “PostgreSQL Unicode(x64)”

Click connect

In the connection dialog type:

Database: <space-name> for example: mySpace

Server: <gateway-endpoint-address> for example: localhost

In Kubernetes it’ll be the load balancer endpoint

port: <data-gateway-bind-port> default: 5432

Choose SSL Mode: disable

Click OK

Click Sign In

To see your schema then on the left panel where the title is "Table" look for the 'search' icon and click on it (next to “Enter table name”), then click twice on the schema you’d like to load from the space.

ODBC Connection to PowerBI using the Data-Gateway

Instructions:

Install ODBC Driver for PostgreSQL (tested on v10.03) to be used by PowerBI.

Launch the “ODBC Data Source Administrator” in your machine

Add DSN data source -> choose PostgreSQL Unicode (x64)

In the connection dialog fill:

Data Source (you can edit the name): “GigaSpaces”

Database: <space-name> for example: mySpace

Server: <data-gateway-endpoint-address> for example: localhost

In Kubernetes it’ll be the load balancer endpoint

port: <gateway-bind-port> default: 5432

Username: admin

Password: 1234

*Username and password aren’t checked at the moment, please write down something

SSL Mode: disable

Click Save

Open PowerBI -> click on “Get Data” -> choose “Other” -> “ODBC” -> Connect

Choose Data source name (DSN) which you created before: “GigaSpaces” -> click “OK”

In the navigator, navigate to the schema of your Space and choose it -> click Load.

PowerBI will load the schema and it’ll be shown in the “Fields” section of your screen.
