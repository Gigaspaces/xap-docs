GigaSpaces Ops Manager

The Ops Manager allows you to perform the basic functions of using an intuitive, GUI-based interface. The IT administrator can:

Connecting to a Data Source — Connect InsightEdge to a traditional SQL database or data warehouse and import the selected data. Use the generated Blueprint to deploy the Space application with its configuration to the InsightEdge data platform.

Monitor my services — Review the real-time performance of the system, and enable an IT administrator or developer to easily monitor the health of all its service types.
Receive real-time availability and resource alerts immediately when they appear.

Analyze my data — enable an IT administrator or developer to review the Space data, drill down to its object types and analyze the data by executing queries using the SQL editor.
