Supported Operating Systems: Any Windows operating system supported by the designated .NET Framework version.

supports Microsoft .NET Framework only. It does not support Microsoft .NET Core.

Installation

is packaged as a standard Windows Installer package (.msi file). To start the installation simply double-click the msi file, and the installation wizard will pop up and guide you through the installation process. After you accept the license agreement, you will see the standard installation details screen, allowing you to modify the installation path (default is C:\GigaSpaces\XAP.NET [%=Versions.product-version-short%] x64) and which features to install.
