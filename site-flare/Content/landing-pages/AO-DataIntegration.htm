Smart DIH

Data Integration (DI)

The Data Integration (DI) layer is a vital part of the Smart DIH platform. It is responsible for a wide range of data integration tasks such as ingesting data in batches or streaming data changes. This is performed in real-time from various sources and systems of record (SoR). The data then resides in the In-Memory Data Grid (IMDG), or Space, of the Smart DIH platform.
