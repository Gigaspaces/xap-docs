Kubernetes

and Smart DIH solutions use Kubernetes as an orchestration solution and as an enabler for using advanced technologies such as memory grid, streaming and automation. Using Kubernetes, our solutions provide high quality in the areas of:

  • Network and compute control

  • High availability and self-healing

  • Security

  • TCO reduction and maintainability

Our installations using Helm charts ensures that the deployment of our solution is easier for operational staff to manage it. This is also due to the fact that best practices and standard Kubernetes procedures are followed. The installation is highly flexible and can be tailored to user needs by via a rich set of deployment parameters.

It is also possible to deploy a custom GigaSpaces processing unit (PU) in a Kubernetes environment. The procedure for this and other Kubernetes procedures, both for Smart Cache and Smart DIH, are described in our Installation Guide.
