Getting Started with GigaSpaces

A full list of sample projects and solutions can be found here (requires a login to the Support portal).

Once you have downloaded GigaSpaces, a list of examples can be found in the <gs-home>/examples directory.

Examples can also be run using the <gs-home>/bin/gs.bat or gs.sh command.

The gs demo command

The first example using GigaSpaces will run the software

If you have never used GigaSpaces before, this is the starting point to understand the software environment and capabilities.

This introduction consists of four parts:

  • Part 1 — Setting the Environment — Prepare a database that will be imported into GigaSpaces. In these examples, we will use data from the PostgreSQL database.

  • Part 2 — Launch Ops Manager — Run and view the GigaSpaces user interface.

  • Part 3 — Import a Database — Bring data from the PostgreSQL database into Smart Cache, the GigaSpaces in-memory grid, and run some SQL operations.

  • Part 4 — Convert a Program — Explore the tools to convert a Java program from JDBC access to the GigaSpaces Java API.
