DI Installation Layer: Prerequisites

There is a list of prerequisites that have to be completed prior to the installation of the DI layer.

  • Component Deployment: DI layer components can be deployed in a Kubernetes environment only!

  • Resources: Minimal resources required for a Kubernetes cluster to host the DI layer: 32GB RAM, 16 cores, 100GB disk space

  • On-Prem to Cloud Connection: VPN between on-prem source database and DI Kubernetes cloud segment
