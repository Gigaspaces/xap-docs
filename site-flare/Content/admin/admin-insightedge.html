Administering

In addition to monitoring and maintaining the core, there are administration issues that are specific to Apache Spark and . This section describes how to set up on a cluster, and provides information about logging and how to modify the default logging configuration. The REST Manager API also provides the following administrative capabilities:

  • Deploying an application
  • Monitoring the status of Apache Spark jobs
  • Monitoring applications
  • Updating application files
  • Terminating an Apache Spark job
  • Deleting an application
