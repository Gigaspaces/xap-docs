Scala

GigaSpaces offer several extensions to the API, to more easily combine Scala with . This includes support for constructor-based properties, a wrapper around the API for task execution, predicate-based queries, and a scripting executor (part of the Dynamic Language Tasks feature).

This section also provides an enhanced REPL demo of what can be done with the Scala shell, along with a sample project that shows how Scala can be used in a real project, and how Scala and Java code can be integrated.

The minimum Scala version supported by the Scala extension is [%=Versions.scala%].

Assuming there is a scala installation under $SCALA_HOME, the jars under $SCALA_HOME/lib should be copied to $GS_HOME/lib/optional/scala/lib.
