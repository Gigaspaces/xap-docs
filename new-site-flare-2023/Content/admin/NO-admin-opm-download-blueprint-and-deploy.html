Download Service Blueprint and Deploy

To download and deploy the service blueprint, click Download Service Blueprint This creates a zip file with scripts and files necessary to deploy the system. These files can be customized as required.

There are two options for the Blueprint:

  • Manual -- Deployment customized by user.
  • Automatic -- Pre-configured for deployment on the user's local machine.

Deploying to the user's local machine

The Service Blueprint zip file is deployed as follows:

  • Unzip the Blueprint in the bin directory of the local installation.
  • At the command prompt, run the following batch command:
    gs.bat host run-agent --auto --gsc=3
  • A browser window will open. Enter the following URL:
    http://localhost:8090

The Ops Manager Landing Page now displays in the local host.
