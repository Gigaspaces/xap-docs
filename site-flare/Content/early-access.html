Early Access

There is currently no early access release available for evaluation.

This page contains early access information for version of products, scheduled for release at the end of Q2 2019. Early access builds are intended for those who want to get involved in the development process and try out new features and functionality early on, and even affect the final outcome. If you have any feedback on early access features, we'd love to hear it!

Early access builds are provided as is, and should not be used in production. The latest stable release is version 14.5.0.

15.0.0 M8 (Aug-12-2019)

Download Links

  • InsightEdge Enterprise
  • XAP Enterprise | XAP.NET Enterprise

Changelog

Features and Enhancements

  • : Kubernetes network policy enhancement to enable data operations from remote clients via a public IP/port.
  • : Added Lucene support for collections in full text search.
  • : Enhance InsightEdge JDBC driver to optionally get schema from an alternative URL instead of from a Space.
  • : Upgrade Jetty to 9.4.19.
  • : Improve SQL query performance when distinct keyword is used.
  • : User-defined SQL functions now support setting parameters.
  • : Added the CONTAINS_KEY built-in SQL function that checks if a map contains a key.
  • : The GS Agent only monitors (and restarts) components that were started successfully.
  • : Upgrade RocksDB to 6.0.1.
  • : Upgrade Hibernate to 5.2.
  • : Added support in AnalyticsXtreme for Hive partitioned tables.

Resolved Issues

  • : GS Agent components don't terminate when the parent process is no longer alive.
  • : Full recovery failed when using a FIFO group.
  • : Polling container didn't have a graceful shutdown when using a DynamicEventTemplate in combination with a SQL query.
  • : Full recovery does not preserve the FIFO grouping order.
