Data Manipulation Syntax

Insert: 

{ INSERT } INTO tablePrimary
[ '(' column [, column ]* ')' ]
[ VALUES '(' column [, column ]* ')' ]

 

Update:

> 
UPDATE tablePrimary
SET assign [, assign ]*
[ WHERE booleanExpression ]

 

Delete:

> 
DELETE FROM tablePrimary [ [ AS ] alias ]
[ WHERE booleanExpression ]

> 
EXPLAIN PLAN
   FOR { query }

   FOR { query }
