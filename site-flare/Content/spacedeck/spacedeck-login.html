SpaceDeck Login

If the environmental variable AUTH=on, the login screen will display to begin using SpaceDeck.

See Managing Users and Roles for a discussion of GigaSpaces security procedures.

Post-Installation First-Time Login - Configuring SSO

The owner (superuser) of the system should initially login via the password option with details provided by as part of the installation.

This step must be completed in order for the superuser to configure SSO (Single sign-on).

Standard User Login (through SSO)

Standard users enter their user credentials to login to Space Deck using SSO (Single Sign-On). Click Login
