Multi-Model Store

GigaSpaces products provide a true multi-model data store that can hold any type of data. The data grid can function as an object store or a document store, and supports a disparate array of query options, such as SQL queries, free text search, geospatial queries and more.

The data grid can store data in object mode, binary mode, or compressed mode and supports both annotation-based and XML-based metadata. You can make querying even faster using one or more of the supported indexing options.
