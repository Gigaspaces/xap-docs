Tableau Integration

This topic describes how to set up an integration of Tableau with in a Microsoft Windows environment, so that the JDBC driver can be used to retrieve data and display it in a graphic visual representation.

Tableau version 2019.1 or later is required.

Integrating Tableau with

In order to integrate Tableau with , you need access to a space that is up and running. After setting up and starting , do the following to create the necessary environment to integrate Tableau and view data:

  1. Download and install the Tableau desktop.

  2. Set up the JDBC connection.

  3. Download and configure Tableau to work with .

Setting Up the JDBC Connection

To configure the connection:

  1. Run <XAP_HOME>\bin\insightedge maven install to install the Maven artifacts.
  2. Run <XAP_HOME>\insightedge\tools\jdbc\build-jdbc-client.cmd to create the insightedge-jdbc-client.jar file.
  3. Copy the file to the Tableau Drivers folder.

  4. Run the following command to start Tableau with the connector: 

    <TABLEAU_HOME>\bin\tableau.exe -DConnectPluginsPath=<GS_HOME>\insightedge\tools\jdbc

Setting Up Tableau

After Tableau has been configured to connect to , you can configure Tableau to read the data.

Configuring the Data Source

To configure the data source:

  1. Start Tableau.

  2. Click Connect in the Tableau desktop, and select Gigaspaces InsightEdge as the data source.

    If on the first access Gigaspaces InsightEdge does not appear in the Connect menu, Click More and select Gigaspaces InsightEdge from the alphabetic list.

  3. In the Gigaspaces InsightEdge window, configure the options as follows:
    1. Enter the Server host name or IP address. Include the port number if the space is not using the default port.
    2. (Optional) If your system includes AnalyticsXtreme, you can type True for Enable AnalyticsXtreme.
    3. Enter the Space name.
    4. (Optional) Add Advanced Parameters for the JDBC connection as key-value pairs.
    5. If your system is configured with a secure Space (See Security), enter the Username and Password.

  4. Click Sign In.

  5. Verify that Tableau can see the Tables in your space.

Querying the Data

Everything that needs to be configured has been set up, and you can execute queries against the data. This section presents a small example to read the data and display it in a graphic visual format. You can then query and see the activity that occurs under the hood when Tableau accesses it as an SQL database and reads the requested data.
