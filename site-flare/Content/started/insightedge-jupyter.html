Using the Jupyter Notebook with PySpark in

This section describes how to integrate Jupyter to be used with PySpark in.

Prerequisites

  • Your host machine must have Python release 2 or 3, including pip, installed

  • Include <python home> in your path.

Incorporating Jupyter

To access PySpark using the Jupyter dashboardl:

  1. Install Jupyter with the <python home>/Scripts/pip install jupytercommand.

  2. Include <pythod-home>/Scripts, in your path.

  3. Add these entries to <XAP_HOME>/bin/insightedge/insightedge-pyspark:

    export PYSPARK_DRIVER_PYTHON=jupyter
export PYSPARK_DRIVER_PYTHON_OPTS='notebook'

Running PySpark from Jupyter Dashboard

Run the command <XAP_HOME>\bin\insightedge\insightedge-pyspark and a browser window opens at localhost:8888 with the Jupyter dashboard. From the dashboard you can view or run existing notebooks or create a new notebook by clicking New and selecting any of the available notebooks. Using the DataFrames API to manipulate data, you can load and access InsightEdge data as shown in the PySpark command line shell.
