JDCB V3  SQL Syntax - Supporting Multiple Statements

statementList: 

statement [ ';' statement ]* [ ';' ]

 

statement: 

|   explain
|   describe
|   query

  • Data Definition (DDL) Syntax

  • Data Manipulation (DML) Syntax

  • Data Query (DQL) Syntax

  • Transaction Control (TCL) Syntax

  • Functions and Operators

  • Data Types
