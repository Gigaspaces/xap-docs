SQL Clients

The InsightEdge JDBC driver can be used with existing visualization tools, to gain insight into the data grid by using the specific tool-provided features such as drag and drop.

Examples:

  • Tableau

  • MicroStrategy

  • QlikView

  • SquirreL

  • SQLLine

SquirreL

To install SquirreL for use with the :

  1. Download and install SquirreL.

  2. Copy all of the JAR files (located in <XAP-HOME>\lib\required) to the squirreL lib folder.

  3. Copy all of the JAR files (<XAP-HOME>\insightedge\lib\jdbc) to the squirreL lib folder.

  4. Start SquirreL.

  5. Click the Drivers vertical tab on the left, and then click + to add a new driver.

  6. Click the Aliases vertical tab on the left, and then click + to add a new connection. Sample URL: jdbc:insightedge:url=jini://*/*/mySpace?locators=127.0.0.1&groups=xap-

  7. Create a connection to the data grid.

  8. Run a test query, for example SELECT * FROM Product; (the data grid was filled with Product objects in advance).
