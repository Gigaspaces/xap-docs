Introducing SpaceDeck

GigaSpaces new User InterfaceGigaSpaces SpaceDeck is now available for Smart DIH users.

SpaceDeck is the ultimate command and control center for the Digital Integration Hub (Smart DIH). It is a secure and fast data ops solution that democratizes your data!

This introductory version does not cover all functionality and is intended as an introduction to the new UI and a supplement to our existing Ops Manager, GigaSpaces Management Center and Web Management Console UIs. SpaceDeck is initially available for SmartDIH installations, and will be extended in future releases to eventually replace the older UIs.

SpaceDeck includes the following features:

  • Hosts - XXXX

  • Spaces: display status, monitoring and resource usage of Spaces and related partitions.

  • Data Sources: specify the data source that will be used to provide data in a data pipeline.

    Data Pipeline: display data pipeline details such as monitoring, source table/s in the System of Record, resulting Object in GigaSpaces, and parameters of the data connection. Also allows quick one-click creation of new data pipelines.

  • Services - XXXX

  • Data Query: Querying the data using SQL commands, supporting full ANSI SQL-99 syntax.

Data Pipeline/Data Integration Enhancements

  • Out of the box integration logic and generic data pipelines dramatically shortens time-to-market for new digital services and minimizes service downtime.

    • Data pipeline support is enhanced to include DB2 on IBM iSeries (AS/400), in addition to previously-supported IBM z/OS mainframe.

SQL Enhancements

  • Document Support in SQL – queries for nested objects can now be accessed via the SQL SELECT statement and the WHERE clause.

 

  • Out of the box integration logic and generic data pipelines dramatically shortens time-to-market for new digital services and minimizes service downtime.

    • IBM Infosphere Data Replication (IIDR) is available as an out-of-the-box connector engine for data pipelines.

    • Data pipeline support is enhanced to include DB2 on IBM z/OS mainframe, Oracle and MS SQL Server.

    • Generic declarative data loader can load any data based on configuration per data source.

    • Ensure business continuity while changing the data schema (adding tables) and reloading data.
