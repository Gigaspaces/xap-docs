Tableau Integration

This topic describes how to integrate Tableau with in a Microsoft Windows environment using the JDBC connector to retrieve data for visualization and business intelligence needs.

Prerequisites

Your host machine must have the following installed:

  • Java 8 or higher
  • Tableau version 2019.1 or higher

Setting up the - Connector for Tableau

The -connector is based on the Tableau Connector SDK and JDBC integration. The setup process includes providing proprietary requirements about various file locations and execution arguments. 's Tableau integration is packaged in the <XAP_HOME>/tools/tableau/gs-tableau.bat script, which automates the setup process and starts Tableau Desktop with the - connector enabled.

The first time this script runs it requires elevated permissions, because it automatically packages and copies the JDBC driver to Tableau's installation files, which generally reside in the Program Files folder.

If necessary you can customize some of the default values, such as the location of the Tableau installation folder or the JDBC driver heap size, by editing the gs-tableau-env script that is also available in the <XAP_HOME>/tools/tableau folder.

Creating a Standalone Connector Package

A standalone connector package is required for machines that access but don't have installed locally. To create the file, run the gs-tableau-package script from the product tools\tableau folder. That file can be distributed and installed on any machine to access data using Tableau.

Connecting to the Space

To configure the data source after Tableau is started:

  1. Click Connect in the Tableau desktop, and select Gigaspaces InsightEdge as the data source.

    If Gigaspaces InsightEdge does not appear in the Connect menu on first access ,click More and select Gigaspaces InsightEdge from the alphabetic list.

  2. In the Gigaspaces InsightEdge window, configure the options as follows:
    1. Type the Server host name or IP address. Include the port number if the Space is not using the default port.
    2. (Optional) If your system includes AnalyticsXtreme, you can type True for Enable AnalyticsXtreme.
    3. Type the Space name.
    4. (Optional) Add Advanced Parameters for the JDBC connection as key-value pairs.
    5. If your system is configured with a secure Space (See Security), type the Username and Password.

  3. Click Sign In to connect to the data grid.

  4. Verify that Tableau can see the Tables in your Space.

    A POJO in the data source must use java.sql.Date rather than java.util.Date.
