On Demand Object Analysis

By performing an Object Analysis, you can determine the memory footprint of object types, properties and indexes running in one or more partitions. Two or more Object Analyses can be compared, to spot trends of increasing or decreasing memory consumption.

Although Object Analysis results are a close approximation of memory usage, some variations may exist between the reported results and the exact amount of memory used.

How Do I Perform an Object Analysis?

An Object Analysis is triggered by pressing the xxx button on the Ops Manager xxx screen, and choosing which partition to analyze.

Because the Object analysis process may consume considerable system resources, it is recommended to run the analysis at an off-peak time, when the system is experiencing relatively low usage.
