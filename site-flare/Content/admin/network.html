Network Configuration

Some of the features described on this page are not part of the open-source edition, and are only available in the licensed editions (starting with ).

This section explains how to configure the network to support your implementation. supports unicast service discovery and multicast where available (for both Linux and Microsoft Windows operating systems).

You can configure the Lookup Service (LUS) to work with both unicast and multicast. is supported over NAT and over firewall, and you can either use the default ports or specify custom ports for the different components and services. can also be configured to work with a specific subset of network cards in environments with multiple network cards.
