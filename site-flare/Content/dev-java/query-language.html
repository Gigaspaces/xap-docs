Query Language

JDBC V1

JDBC V1 is similar in its performance and its coverage to the GigaSpaces API and is used only in our solution.

The analysis Explain Plan can be used for debugging.

JDBC V3

JDBC V3 has SQL 99 coverage, however it is more complex but the performance of simple queries will be slightly less than JDBC V1.

In addition, JDBC V3 has support for ODBC and JDBC Postgress flavors.

Analysis tools such as Explain Plan and Explain Analyze can be used for debugging.

For additional information see the Integrating with BI Tools section.
