SpaceDeck – Configuration File for Data Pipeline

In the screen to define a new data pipeline, the configuration parameters can be entered manually, or can be define in a configuration file. The configuration file is in JSON format.

Sample Data Pipeline Configuration File

The configuration file is in JSON format. A sample configuration file is shown below.


{
    "name": "pipeline",
    "spaceName": "ditest1-ts",
    "sorName": "DB2",
    "cdcProvider": "IIDR",
    "jobConfigs":{
        "CDC": {
            "kafkaTopic": "noi_cdc"
        },
       "INITIAL_LOAD": {
        "kafkaTopic":"noi_il"
        }
  }
}
