Maven

Maven is a tool used to automate and simplify the development cycle of any Java-based project. The plugin for Maven utilizes Maven to simplify the development cycle of -based applications. This section describes the Maven artifacts that are provided, and how to get started with using Maven..

Maven Artifacts

is Maven-friendly. It is built using Maven and is designed to be easily used by developers constructing applications. In this page we'll explain how to use with Maven.

Prerequisites

  • Your host machine must have Maven nstalled.
  • Include the Maven bin directory in the system PATH variable.

Installation

Run the following command:

<XAP_HOME>\insightedge\bin\gs maven install

Maven Repository

artifacts are not currently published in the Maven Central Repo, so you must first configure a repository:

<repository>
   <id>org.openspaces</id>
   <url>http://maven-repository.openspaces.org</url>
</repository>

Artifacts

The main dependency required to use is xap-openspaces.

<dependency>
  <groupId>org.gigaspaces</groupId>
  <artifactId>xap-openspaces</artifactId>
  <version>[%=Versions.maven-version%]</version>
</dependency>
