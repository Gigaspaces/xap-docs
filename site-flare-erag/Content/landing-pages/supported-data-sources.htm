eRAG Supported Data Sources

eRAG provides connectors to various different data sources.

Some of the data sources can be combined with one another (these are specified as “Multi Mode” below), and some can only be used as a single data source.

The table below shows the types of data sources that are supported by eRAG, although additional data sources can be added on-demand, with short notice (depending on the actual data source).
