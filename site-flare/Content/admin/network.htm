Network Configuration

Some of the features described on this page are not part of the open-source edition, and are only available in the licensed editions (starting with XAP Premium).

This section explains how to configure the network to support your XAP implementation. XAP supports unicast service discovery and multicast where available (for both Linux and Microsoft Windows operating systems).

You can configure the Lookup Service (LUS) to work with both unicast and multicast. XAP is supported over NAT and over firewall, and you can either use the default ports or specify custom ports for the different XAP components and services. XAP can also be configured to work with a specific subset of network cards in environments with multiple network cards.
