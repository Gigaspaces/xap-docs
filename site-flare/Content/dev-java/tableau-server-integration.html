Integrating with Tableau Server

This topic describes how to publish from from to Tableau Server, via Tableau desktop, where it can be shared throughout the organization. To integrate with Tableau Server you must:

  • Setup the JDBC Connector on Tableau Server
  • Set up security on the server to work with Tableau Server

Then you can publish data from the desktop to Tableau Server.

Prerequisites

You must setup the Tableau desktop with JDBC Connector for Tableau. Your server must have the following installed:

  • Java 8 or higher

Integrating with Tableau Server

Setting Up the JDBC Connector on Tableau Server

To setup the JDBC Connector:

  1. Copy the JDBC driver from your desktop <TABLEAU_HOME>\Drivers folder to the same folder on the server.

  2. Copy the \tools\tableau\insightedge-connector folder to any folder on the server.

  3. To configure insightedge-connector run the following commands:

    tsm configuration  set -frc -k native_api.connect_plugins_path 
-v C:/<insightedge_connectors folder>
tsm pending-changes apply

  4. To verify (is this command required or only for verification) the configuration run the following command:

    tsm settings export --output-config-file config.txt

and verify that config.txt contains the line native_api.connect_plugins_path -v C:/tableau_connectors

Setting Up the Security

To setup security to access Tableau Server

  • Add the following to setenv-overrides.bat/.sh
.export XAP_OPTIONS_EXT="-Dcom.gs.security.enabled=true -Dcom.gs.manager.rest.ssl.enabled=false"
set GS_OPTIONS_EXT="-Dcom.gs.security.enabled=true -Dcom.gs.manager.rest.ssl.enabled=false"
  • Create an new user with all permissions. This user will be used to connect to Tableau Server.

Publishing to Tableau Server

After the setup process is complete and the connection has been configured, you can publish to the server following these steps

  1. Connect to the space.

  1. Create a sheet corresponding to your data.

  1. Select Sign Infrom the Server menu to sign in to your Tableau Server.
  2. Select Publish Data Source from the Server menu and to select your sheet to publish.
  3. Enter any of the options in the Publish Data Source window and click Publish.

The data source is published to Tableau Server and the server URL is opened in your browser. Log in and verify that the data is published on the server.
