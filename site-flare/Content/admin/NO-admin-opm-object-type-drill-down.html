Object Type Drill Down screen

From the list of Object Types, click an Object Type to see the Object Type Drill Down screen.

General information

The top part of the page contains the following information:

  • Name of Object Type
  • Class name -- xxxxxxxxxxx
  • Entries -- xxxxxxxxxxxxxxxxxx
  • Entries -- xxxxxxxxxxxxxxxxxx
  • Notify Containers -- The number of notifications generated by the Object Type (see xxxxx).

Graphical representation of Space activity

Entries by partition shows the distribution of the Object Type data entries across the available primary partitions.

Reads by partition shows similar graphs representing the distribution of data read activity for the Object Type, across the primary partitions.

List of Object Type properties

This section of the screen lists the properties found in the Object Type. There are three special icons that may be displayed next to a field:

  • Primary key
  • Index
  • Routing key -- a hash total used in partition load balancing

List of Object Type indexes

Five types of indexes are available:

  • Regular
  • Composite
  • Nested
  • Text
  • Geolocation

Next to each index is the list of one or more properties used to create the index.

 

SQL Editor

This version of the SQL Editor is predefined in the scope of the Object Type.
