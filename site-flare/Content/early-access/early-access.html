There is currently no early access release available for evaluation.

This page contains early access information for the platform and data grid version , which is scheduled for release at the end of Q2 2019. Early access builds are intended for those who want to get involved in the development process and try out new features and functionality early on, and even affect the final outcome. If you have any feedback on early access features, we'd love to hear it!

Early access builds are provided as is, and should not be used in production. The latest stable release is version 14.2.0.
