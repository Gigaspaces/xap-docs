SpaceDeck – Data Pipeline Overview

Data Pipelines are flexible, configurable, no-code data channels that gather data from a mix of System of Record databases and tables. This data is loaded into the extremely high speed GigaSpaces data grid, where it can be accessed on a read/write basis.

Advantages of Data Pipelines include:

  • Flexible extract of data from multiple databases and tables

  • Import of data into the high-speed GigaSpaces in-memory

  • Unified data model for read/write of pipeline data
