IIDR Oracle Agent Installation

For deployment model option 1, the IIDR source database agent should be installed on a source database server.

The following instructions should be followed to complete the installation for the IIDR Oracle Agent.

 

  1. Login to the Oracle server mcahing as gsods OS user

  2. Create a directory where the IIDR Agent software will be installed.

    3. mkdir -p /home/gsods/iidr
cd /home/gsods/iidr

  3. Download the IIDR Oracle Agent to the newly created directory of /home/gsods/iidr

  4. The downloaded file should then be used in the setup using the following command (example Agent version used is 11.4.0.4:

    5. setup -iidr-11.4.0.4-5629-linux-x86.bin

  5. Ensure that the DISPLAY environment is not set

    6. unset DISPLAY

  6. Begin the IIDR Oracle installation:

    7. chmod u+x setup-iidr-11.4.0.4-5629-linux-x86.bin
./setup-iidr-11.4.0.4-5629-linux-x86.bin

  7. Select the Locale (language)
