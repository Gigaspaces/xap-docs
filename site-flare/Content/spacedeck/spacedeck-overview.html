GigaSpaces SpaceDeck Overview

SpaceDeck is a user-friendly web interface that will provide full functionality and control of the GigaSpaces environment.

This introductory version does not cover all functionality and is intended as an introduction to the new UI and a supplement to our existing Ops Manager, GigaSpaces Management Center and Web Management Console UIs. SpaceDeck is initially available for SmartDIH installations, and will be extended in future releases to eventually replace the older UIs.

Using SpaceDeck, the user can define the following items:

  • Data Source – the database in the System of Record that will be used as a source and target of data for the data pipeline

  • Data Pipeline – the mechanism transfer data to and from selected tables in the data source to selected Space objects in teh GigaSpace in-mempory data grid

SpaceDeck also provide a Spaces Status screen that shows status and real-time monitoring of the Spaces in the data grid.
