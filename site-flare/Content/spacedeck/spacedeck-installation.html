SpaceDeck Installation

Download SpaceDeck

Contact your GigaSpaces representative for instructions how to download the Docker image for SpaceDeck.

Sample for docker-compose File


version: ‘2’
services
  space-deck:
    image: latest (e.g. gigaspaces/spacedeck:1.0.2)
    ports:
     - 4200:3000
    container_name: space-deck
    environment:
     - API_URL=http://di:8090/v2
     - DI_API_URL=http://di:6080
     - DI_MDN_API_URL=http://di:6081
     - AUTH= on
-IDLE_SESSION_TIMEOUT = 60
    extra_hosts:
     - “di:172.31.47.51”

Environmental Variables

AUTH - set to on to enable authentication.

API_URL - set to the url of the api (ie. http://localhost:8090/v2).

DI_API_URL - set to the url of the di api (ie. http://localhost:8090/v2).

DI_MDN_API_URL - set to the url of the mdn api (ie. http://localhost:8090/v2).

IDLE_SESSION_TIMEOUT - how long idle time in minutes before logging out, default 30.
