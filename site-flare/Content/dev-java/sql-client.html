 SquirreL

To install SquirreL for use with the InsightEdge platform:

  1. Download and install SquirreL.
  2. Copy all of the XAP JAR files (located in <XAP-HOME>\lib\required) to the squirreL lib folder.
  3. Copy all of the InsightEdge JAR files (<XAP-HOME>\insightedge\lib\jdbc) to the squirreL lib folder.
  4. Start SquirreL.

  5. Click the Drivers vertical tab on the left, and then click + to add a new driver.

  6. Click the Aliases vertical tab on the left, and then click + to add a new connection. Sample URL: jdbc:insightedge:url=jini://*/*/mySpace?locators=127.0.0.1&groups=xap-14.0.0

  7. Create a connection to the XAP data grid.

  8. Run a test query, for example SELECT * FROM Product; (the data grid was filled with Product objects in advance). ```
