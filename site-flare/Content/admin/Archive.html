The following archival documentation contains legacy content for older Smart Cache software releases that are no longer supported. This information is provided for reference purposes only.

Documentation for Smart Cache versions prior to version 12.2 is no longer available online. If you are looking for documentation for one of those versions, download the relevant archive from the table below and browse it offline. Simply unzip the file and open the main index.html file.
