Smart DIH

SpaceDeck - SpaceQL

SpaceQL is a GraphQL user interface, integrated in the DIH SpaceDeck. It allows to perform interactive GraphQL queries.

For additional information, refer to the DI GraphQL page.

Main Screen

Access the SpaceQL screen directly from the SpaceDeck menu.

Browse GraphQL Schema in SpaceQL

To browse GraphQL schema in SpaceQL, click Docs. All schema types are shown.

Query the GraphQL Service

You can query the GraphQL service through SpaceQL, as above, or using the dihctl application. For additional information, refer to the DI GraphQL page.
