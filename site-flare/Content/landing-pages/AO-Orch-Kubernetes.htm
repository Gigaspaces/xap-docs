Kubernetes

Our Smart DIH uses Kuberenetes as its orchestration solution and as an enabler for using advanced technologies such as memory grid, streaming and automation. Using Kubernetes, Smart DIH provides high quality in the areas of:

  • Network and compute control

  • High availability and self-healing

  • Security

  • TCO reduction and maintainability

Smart DIH installation using Helm charts ensures that the deployment of our solution is easier for operational staff to manage it. This is also due to the fact that best practices and standard Kubernetes procedures are followed. The installation is highly flexible and can be tailored to user needs by via a rich set of deployment parameters.
